Crystals Collector!


Rules:


A random number will be given at the start of the game. There are four crystals below. By clicking on a crystal a specific amount of points will be added to the total score.


Win the game by matching the total score to the random number.

Lose the game when the total score goes above the random number.

The value of each crystal is hidden until it is clicked on.


Each time the game starts, the game will change the values of each crystal.
